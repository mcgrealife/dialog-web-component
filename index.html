Default HtmlDialog vs Extended HtmlDialog

This is a default HtmlDialog Element

Baseline available since March 2022

Benefits:
  • autofocus
  • esc to close
  • Returns focus after closing
  • Always on top, without z-index (or portals!)
  • backdrop is styled via ::backdrop pseudo-selector
Limitations:
  • Scrolls content behind
  • Does not close on click outside
